### Проект интернет-магазина

 - Инициализирован проект Django.
 - Создано приложение Catalog, зарегистрировано, настроена маршрутизация.
 - Созданы HTML-шаблоны home.html и contacts.html, для стилизации страниц использован Bootstrap.
 - Созданы контроллеры для отображения HTML-шаблонов, настроена маршрутизация.
 - Реализована форма обратной связи на странице контактов. Настроено отображение сообщения об успешной отправке данных. Так же, отправленные данные выводятся в консоль.
 - Конфиденциальные данные сокрыты в .env.
